# OptiFlow 🚀 — Premium Media Optimizer

## 🇮🇩 Bahasa Indonesia

**OptiFlow** adalah aplikasi pengoptimal media (gambar & video) berbasis web yang modern, cepat, dan mengutamakan privasi. Seluruh proses kompresi dilakukan secara lokal di browser Anda menggunakan teknologi WebAssembly, sehingga file Anda tidak pernah keluar dari perangkat Anda.

### ✨ Fitur Utama
- **Kompresi Gambar**: Mengurangi ukuran file JPG, PNG, dan WEBP hingga 90% tanpa kehilangan kualitas visual yang signifikan.
- **Kompresi Video**: Didukung oleh FFmpeg.wasm untuk transcoding video MP4/MOV langsung di browser.
- **Privasi 100%**: Tidak ada server yang terlibat. Data Anda tetap di tangan Anda.
- **UI Premium**: Desain *Glassmorphism* yang responsif dan interaktif menggunakan React + Tailwind CSS v4.
- **Kontrol Kustom**: Atur tingkat kualitas sesuai keinginan Anda (10% - 100%).

### 🛠️ Teknologi yang Digunakan
- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS v4 + Framer Motion
- **Core Engine**: 
  - `FFmpeg.wasm` (Video)
  - `browser-image-compression` (Gambar)
- **Icons**: Lucide React

### 🚀 Cara Menjalankan Secara Lokal
```bash
# Clone repository ini (jika ada)
# Buka folder project
cd optiflow

# Install dependensi
npm install

# Jalankan server pengembangan
npm run dev
```

## 🇺🇸 English

**OptiFlow** is a modern, fast, and privacy-first web-based media optimizer for images and videos. The entire compression process happens locally in your browser using WebAssembly technology, ensuring your files never leave your device.

### ✨ Key Features
- **Image Compression**: Reduce JPG, PNG, and WEBP file sizes by up to 90% while maintaining stunning visual clarity.
- **Video Compression**: Powered by FFmpeg.wasm for MP4/MOV video transcoding directly in the browser.
- **100% Privacy**: No servers involved. Your data stays in your hands.
- **Premium UI**: Responsive and interactive *Glassmorphism* design using React + Tailwind CSS v4.
- **Custom Controls**: Adjust quality levels to your liking (10% - 100%).

### 🛠️ Tech Stack
- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS v4 + Framer Motion
- **Core Engine**: 
  - `FFmpeg.wasm` (Video)
  - `browser-image-compression` (Image)
- **Icons**: Lucide React

### 🚀 Local Setup
```bash
# Clone this repository
# Enter the project directory
cd optiflow

# Install dependencies
npm install

# Run the development server
npm run dev
```
